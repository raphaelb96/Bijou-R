PHASE 1 — DOMAINE ET EMAILS (Jour 1 — 2 heures)
================================================================================

ÉTAPE 1.1 — Acheter le domaine bijou-r.com
───────────────────────────────────────────
Durée : 15 minutes
Où : va sur namecheap.com
Comment :
  1. Cherche "bijou-r.com" dans la barre de recherche
  2. S'il est disponible, ajoute au panier
  3. Achète aussi "bijou-r.fr" et "bijou-r.co.il" si disponibles (~10$/an chacun)
  4. Paye avec carte ou PayPal
  5. Active "WhoisGuard" (protection vie privée) — gratuit chez Namecheap
Résultat attendu : tu reçois un email de confirmation. Tu as le domaine.

REMARQUE : si bijou-r.com est pris, essaie bijour.com ou bijou-r.co.
Préviens avant de choisir une alternative.

ÉTAPE 1.2 — Créer les emails pro
──────────────────────────────────
Durée : 30 minutes
Où : workspace.google.com
Comment :
  1. Clique "Commencer" → choisir le plan "Business Starter" (6$/mois/user)
  2. Entre "bijou-r.com" comme domaine
  3. Vérifie le domaine chez Namecheap (copier-coller les enregistrements DNS
     que Google te donne dans le panneau Namecheap → "Advanced DNS")
  4. Crée ces 4 adresses :
     - hello@bijou-r.com
     - commandes@bijou-r.com
     - support@bijou-r.com
     - pro@bijou-r.com
  5. Teste en t'envoyant un email depuis chaque adresse
Résultat attendu : 4 boîtes mail pro fonctionnelles.

PHASE 2 — SHOPIFY (Jour 2-3 — environ 8 heures)
================================================================================

ÉTAPE 2.1 — Ouvrir la boutique Shopify
────────────────────────────────────────
Durée : 20 minutes
Où : shopify.com
Comment :
  1. Clique "Démarrer l'essai gratuit"
  2. Nom de boutique : "Bijou-R"
  3. Email : hello@bijou-r.com
  4. Remplis le questionnaire (type de produit : bijoux / accessoires)
  5. Plan : choisis "Advanced" (240€/mois) — ne prends pas Basic,
     tu as besoin de Shopify Flow et du multi-marché
Résultat attendu : ta boutique est créée, tu arrives sur le dashboard.

ÉTAPE 2.2 — Connecter le domaine
──────────────────────────────────
Durée : 20 minutes
Où : Shopify → Paramètres → Domaines
Comment :
  1. Clique "Connecter un domaine existant"
  2. Suis les instructions pour modifier les DNS chez Namecheap
  3. Attends 10-30 minutes pour la propagation DNS
  4. Vérifie que "bijou-r.com" apparaît en vert dans Shopify
Résultat attendu : ton domaine pointe vers ta boutique.

ÉTAPE 2.3 — Acheter et installer le thème
──────────────────────────────────────────
Durée : 30 minutes
Où : Shopify Theme Store
Thème recommandé :
  → "Impulse" de Archetype Themes (380$) — le meilleur pour la bijouterie
  → Ou "Prestige" de Maestrooo (320$) si tu veux plus classique

  Si budget serré pour l'instant : Dawn (gratuit) pour commencer,
  tu migres vers Impulse quand les premières ventes arrivent.

Après installation :
  1. Palette couleurs :
     - Couleur principale : #C9A961 (or champagne)
     - Fond : #FAF7F2 (blanc cassé)
     - Texte : #0A0A0A (noir profond)
     - Accent : #C9A961
  2. Polices :
     - Titres : Cormorant Garamond (disponible dans Shopify)
     - Corps : Inter
  3. Logo : si tu n'en as pas, utilise un texte stylisé "Bijou-R" en
     Cormorant Garamond pour l'instant
Résultat attendu : boutique avec couleurs et polices en place.

ÉTAPE 2.4 — Configurer les informations de base
──────────────────────────────────────────────────
Durée : 20 minutes
Où : Shopify → Paramètres → Général
À remplir :
  - Nom de boutique : Bijou-R
  - Email de contact : hello@bijou-r.com
  - Adresse de l'entreprise (nécessaire pour les factures)
  - Fuseau horaire : Paris OU Jerusalem selon marché principal
  - Devise : EUR (ou ILS pour Israël)
Résultat attendu : les infos de base sont complètes.

ÉTAPE 2.5 — Configurer les paiements
──────────────────────────────────────
Durée : 45 minutes
Où : Shopify → Paramètres → Paiements

Dans l'ordre :
  1. Shopify Payments (si en France) → donne tes infos bancaires
  2. PayPal Express → connecte ton compte PayPal pro
  3. Alma (paiement 3x/4x) → va sur getalma.eu, crée un compte marchand,
     installe l'app Shopify depuis l'App Store Shopify
  4. Klarna → similaire, klarna.com/merchants
  5. Pour Israël : Stripe → crée un compte sur stripe.com,
     connecte via Shopify → Paiements → Fournisseurs tiers

IMPORTANT : teste TOUS les paiements avec une vraie commande test
(remboursée après) avant le lancement.
Résultat attendu : au moins 3 modes de paiement actifs.

ÉTAPE 2.6 — Créer les metafields produits
──────────────────────────────────────────
Durée : 30 minutes
Où : Shopify → Paramètres → Métachamps personnalisés → Produits

Crée ces champs un par un :

  - Champ "carats_or" → Type : Nombre entier
  - Champ "pierre_principale" → Type : Texte court
  - Champ "carat_pierre" → Type : Nombre décimal
  - Champ "purete_diamant" → Type : Texte court
  - Champ "couleur_diamant" → Type : Texte court
  - Champ "certification" → Type : Fichier
  - Champ "poids_grammes" → Type : Nombre décimal
  - Champ "origine" → Type : Texte court
  - Champ "delai_fabrication" → Type : Nombre entier
  - Champ "modele_3d" → Type : Fichier (auto-rempli par pipeline Meshy)
  - Champ "story" → Type : Texte multiligne
Résultat attendu : 12 metafields créés, ils apparaissent sur les fiches produit.

ÉTAPE 2.7 — Créer les pages essentielles
──────────────────────────────────────────
Durée : 3-4 heures (à étaler sur 2 jours)
Où : Shopify → Contenu en ligne → Pages

Créer dans cet ordre (les plus importantes d'abord) :
  1. /pages/livraison-retours
     Contenu minimum :
     - FR : livraison offerte dès 150€, délai 48h, transporteur Colissimo/DHL
     - IL : DHL Express 3-5 jours ouvrés, tarif affiché
     - Retours : 30 jours, gratuit, remboursement sous 5j
     - Contact SAV : support@bijou-r.com + WhatsApp

  2. /pages/garantie-vie
     Contenu :
     - Ce qui est couvert : sertissage, finitions, fermeture
     - Ce qui n'est pas couvert : perte, vol, choc fort
     - Comment activer : formulaire ou email
     - Formulaire de demande de réparation

  3. /pages/a-propos
     Contenu :
     - Histoire de la marque (2-3 paragraphes émotionnels)
     - "Pourquoi Bijou-R existe" (raison d'être)
     - L'atelier, le processus
     - La promesse

  4. /pages/faq
     Au moins 20 questions avec réponses.
     Catégories à couvrir :
     → Commande : "Comment puis-je modifier ma commande ?"
     → Livraison : "Combien de temps pour recevoir ma commande ?"
     → Produits : "L'or 18 carats est-il résistant ?"
     → Tailles : "Comment connaître ma taille de bague ?"
     → Garantie : "Que couvre la garantie à vie ?"
     → Retours : "Puis-je retourner un bijou gravé ?"
     → Paiement : "Puis-je payer en 3 fois ?"
     → Diamants : "Vos diamants sont-ils certifiés GIA ?"

  5. /pages/guide-tailles
     - Tableau tailles bagues (numéro FR / mesure mm / équivalent US/UK)
     - Tableau tailles bracelets (longueur en cm)
     - Tailles chaînes (40/42/45/50 cm)
     - Instructions pour mesurer son doigt à la maison
     - Mention "En cas de doute, prendre la taille au-dessus"

  6. /pages/contact
     - Formulaire de contact natif Shopify
     - Email : hello@bijou-r.com
     - WhatsApp (numéro avec lien wa.me)
     - Horaires de réponse : lun-ven 9h-18h

  7. Pages légales (générateur Shopify) :
     - /pages/politique-confidentialite
     - /pages/cgv
     - /pages/mentions-legales
     - /pages/politique-retours
     - /pages/politique-cookies

Résultat attendu : toutes les pages légales et pratiques créées.

ÉTAPE 5.5 — Créer le workflow Meshy dans n8n
─────────────────────────────────────────────
Durée : 2 heures
Ce workflow : quand tu crées un produit Shopify avec le tag "needs-3d",
il génère automatiquement un modèle 3D et l'uploade sur le produit.

Dans n8n, crée un nouveau workflow et ajoute ces nodes dans l'ordre :

  NODE 1 : Shopify Trigger
  → Event : "Product Created"
  → Credentials : connecte ton shop Shopify
    (Settings Shopify → Apps → Développer des apps → Créer une app →
    Permissions API Admin → Products : lecture + écriture + médias)

  NODE 2 : IF (condition)
  → Vérifie si le product a le tag "needs-3d"
  → Condition : {{$json.tags.includes("needs-3d")}} = true

  NODE 3 : HTTP Request (récupérer la première image du produit)
  → URL : https://{{shopDomain}}/admin/api/2026-01/products/{{$json.id}}/images.json
  → Headers : X-Shopify-Access-Token: {{ton_token}}

  NODE 4 : HTTP Request (Meshy API - créer le job 3D)
  → Method : POST
  → URL : https://api.meshy.ai/v2/image-to-3d
  → Headers : Authorization: Bearer {{ton_meshy_api_key}}
  → Body (JSON) :
    {
      "image_url": "{{URL de la 1ère image du produit}}",
      "enable_pbr": true,
      "surface_mode": "hard",
      "topology": "quad",
      "target_polycount": 30000
    }
  → Récupère le task_id dans la réponse

  NODE 5 : Wait (5 minutes)
  → Pause pendant que Meshy génère le modèle

  NODE 6 : HTTP Request (vérifier le status)
  → Method : GET
  → URL : https://api.meshy.ai/v2/image-to-3d/{{task_id}}
  → Si status = "succeeded" → continuer
  → Si status = "pending" → ajouter un autre Wait de 5 min et revérifier

  NODE 7 : HTTP Request (télécharger le .glb)
  → GET l'URL du .glb depuis la réponse Meshy

  NODE 8 : HTTP Request (upload vers Shopify)
  → POST /admin/api/2026-01/products/{{product_id}}/media.json
  → Body :
    {
      "media": {
        "original_source": "{{glb_url}}",
        "media_type": "MODEL_3D",
        "alt": "Vue 3D - {{product_title}}"
      }
    }

  NODE 9 : Slack (notification)
  → Message : "Modèle 3D généré et uploadé pour {{product_title}} ✅"

  NODE 10 : Shopify (retirer le tag "needs-3d", ajouter "3d-done")

Résultat attendu : workflow automatique qui génère des .glb à chaque nouveau produit.

CONSEIL : teste d'abord manuellement, puis active le trigger automatique.

SUIVI HEBDOMADAIRE — CE QU'IL FAUT VÉRIFIER CHAQUE SEMAINE
================================================================================

Chaque lundi matin, vérifier ces 10 KPIs :

  1. CHIFFRE D'AFFAIRES HEBDO
     Où : Shopify → Analytics → Vue d'ensemble
     Objectif M1 : 5 000-10 000€/semaine

  2. TAUX DE CONVERSION
     Où : Shopify Analytics ou Google Analytics 4
     Objectif : > 1.5% (2% = excellent pour bijouterie)

  3. ROAS META ADS
     Où : Meta Ads Manager → Vue d'ensemble compte
     Objectif : > 2.5 (si < 1.5, pause et nouveaux créatifs)

  4. ROAS GOOGLE ADS
     Où : Google Ads → Vue d'ensemble
     Objectif : > 3.5 sur Search, > 4 sur Shopping

  5. ABANDON PANIER
     Où : Shopify Analytics → Checkout
     Objectif : < 70% (si > 75%, audit le checkout)

  6. FLOWS KLAVIYO
     Où : Klaviyo → Flows → Métriques
     Vérifier : taux d'ouverture (objectif >30%), CTR (>3%), revenu par email

  7. AVIS CLIENTS
     Où : Loox dashboard
     Objectif : minimum 5 nouveaux avis par semaine
     Si 0 avis : vérifier que le flow "Review Request" est actif

  8. STOCK
     Où : Shopify → Produits → Inventaire
     Action : réapprovisionner si < 5 unités sur les bestsellers

  9. TICKETS SAV OUVERTS
     Où : Gorgias
     Objectif : aucun ticket ouvert > 24h

  10. FOLLOWERS INSTAGRAM + TIKTOK
      Croissance hebdo attendue M1 : +100-300/semaine

Chaque vendredi, envoie un résumé de ces 10 métriques à toi-même par email.
Avec le temps, tu verras les tendances. C'est comme ça qu'on prend les bonnes décisions.

RAPPEL — CE QUI FAIT LA DIFFÉRENCE ENTRE UN SITE ORDINAIRE ET UN BANGER
================================================================================

1. L'ANCRE PSYCHOLOGIQUE
   Le bracelet tennis à 24 990€ n'est pas là pour vendre.
   Il est là pour que le 2 490€ paraisse accessible.
   Ne l'enlève jamais de la homepage.

2. LE VIEWER 3D
   C'est le seul endroit sur internet où le client peut "toucher" ton bijou
   avant de l'acheter. Chaque produit doit l'avoir.

3. L'ÉMOTION DANS CHAQUE MOT
   "Elle est pour moi" au lieu de "Ajouter au panier".
   "Finaliser mon écrin" au lieu de "Payer".
   Ces 3 mots font la différence entre une vente et un abandon.

4. LE CONFIGURATEUR
   C'est ta machine à tickets élevés. Une bague sur mesure ne se retourne pas.
   C'est 4 500-15 000€ de ticket moyen contre 1 200€ en catalogue.

5. LE BLOG AEO
   Dans 12 mois, quand quelqu'un demande à Perplexity ou ChatGPT
   "où acheter un bracelet tennis en France", Bijou-R doit être cité.
   Ça ne s'improvise pas. Ça se construit article par article, dès maintenant.
